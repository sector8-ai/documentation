Use this path when your application runs on Python and you want the fastest production onboarding flow.

## Prerequisites

- Python 3.10 or later
- `SECTOR8_API_KEY`
- `SECTOR8_CLIENT_ID`

## Install the SDK

```bash
pip install sector8-sdk
```

Source repository: `https://github.com/sector8-ai/sector8-sdk-python`

## Set your credentials

```bash
export SECTOR8_API_KEY=your_api_key
export SECTOR8_CLIENT_ID=your_client_id
```

## Send your first event

```bash
python - <<'PY'
import asyncio
import os
import sector8

async def main():
    client = sector8.setup(
        api_key=os.environ["SECTOR8_API_KEY"],
        client_id=os.environ["SECTOR8_CLIENT_ID"],
        debug=True,
    )
    result = await client.save_telemetry(
        provider="openai",
        model="gpt-4",
        prompt="Sector8 Python SDK onboarding test",
        completion="Telemetry sent successfully",
        tokens_used=24,
        latency_ms=120,
        cost=0.001,
        success=True,
        metadata={"source": "python-sdk-onboard"},
    )
    print(result["status"])
    await client.close()

asyncio.run(main())
PY
```

Expected output:

```text
success
```

<Tip>
The Python SDK defaults to the production Guard Module at `https://sdkapi.sector8.ai`. Override `base_url` only for staging or custom environments.
</Tip>

## Use the three-line setup

```python
import sector8

client = sector8.setup(
    api_key="your_api_key",
    client_id="your_client_id",
)

client.log_llm_call("openai", "gpt-4", tokens=150, cost=0.003)
```

## Use the async API directly

```python
import asyncio
import sector8

async def main():
    client = sector8.setup(
        api_key="your_api_key",
        client_id="your_client_id",
    )
    await client.save_telemetry(
        provider="openai",
        model="gpt-4",
        prompt="hello",
        completion="world",
        tokens_used=10,
        latency_ms=80,
        cost=0.001,
        success=True,
    )
    await client.close()

asyncio.run(main())
```

## Verify the integration

1. Check production health:

```bash
curl https://sdkapi.sector8.ai/api/health
```

2. Confirm your script returned `success`
3. Confirm the event appears in Sector8 telemetry or the dashboard

<Check>
You are done when your Python script returns `success` and you can see the event in Sector8.
</Check>

## Common mistakes

- Missing `SECTOR8_CLIENT_ID`
- Forgetting to `await client.save_telemetry(...)`
- Overriding `base_url` to an older host
